Authenticket(가칭)
==================

# 1. 목표
공연 및 문화 관람 티켓을 Smart contract로 구현

# 2. 개인키 거래 방지 방안
사용한 티켓은 고객의 개인키에 귀속된 채로 있게된다. 이점을 활용해 레벨링 시스템을 도입하고, 향후 티켓 구매 시 우선순위로 활용한다.
